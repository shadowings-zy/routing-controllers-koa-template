# README

## Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Compiles and hot-reloads for production
```
npm run build:watch
```

### Compiles and hot-reloads for development
```
npm run start:watch
```

### Tips
You need run `npm run build` first, then you can run `npm run start:watch`
